# Task 2 – Account Module

## Overview
This project implements a simple Account class in Java that allows deposits and withdrawals with validation.

## BigDecimal
BigDecimal is used to represent money to avoid rounding errors with float or double.

## Sample Output
Deposited: 1000.00. New balance: 6000.00
Withdrawn: 2000.00. Remaining balance: 4000.00
Final balance: 4000.00
